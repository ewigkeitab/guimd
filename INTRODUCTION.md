# Introduction to Guimd

**Guimd** is a free Markdown editor designed specifically for writing and reading. Simple and lightweight, with no complex features, it offers only a clear and concise interface, allowing you to focus on writing and editing.

## Why use Guimd?

### 1. What You See Is What You Get (WYSIWYG)
No more looking at confusing symbols or weird code, nor do you need to memorize Markdown syntax. After copying and pasting, you won't see a bunch of strange symbols.

### 2. Cross-Platform App
Available on macOS, Windows, and Linux. There's no need to look for similar but different editors across different operating systems.

### 3. Lightweight and Low Overhead
Small in size and fast in execution speed. Opening multiple files at the same time won't cause lag, and it doesn't consume system resources.

### 4. Document Outline
Clickable structured index for seamless document navigation via instant visual search targeting.

## User Operation Instructions

### 1. File Management
- **New File**: Create a new empty document.
- **Open**: Open an existing `.md` file.
- **Save / Save As**: Save your current work. Guimd will prompt you if you try to close with unsaved changes.
- **Recent Files**: Quickly access your recently edited documents from the File menu.

### 2. Formatting Text
Use the minimalist toolbar at the top:
- **Style**: Bold, Italic, and Underline.
- **Structure**: Apply H1, H2, or H3 headings.
- **Lists**: Bullet lists, numbered lists, and task lists (checklists).
- **Advanced**: Blockquotes, code blocks, and tables.
- **Table Controls**: When you click inside a table, additional icons will appear to add/delete rows and columns.
- **Zooming**: Use the **A+** and **A-** icons to quickly adjust the editor font size.
- **View Mode**: Click the **Eye** icon to toggle between Edit Mode and a distraction-free View Mode.

### 3. Links and Images
- **Inserting Links**: Click the Link icon. You can specify both the **Display Text** and the **URL**. If you select text first, it will be pre-filled.
- **Inserting Images**: Click the Image icon and provide the URL or local path to the image.

### 4. Navigation and Search
- **Document Structure**: Click the "**Index**" button at the bottom right to see the heading outline. Clicking a heading will scroll the editor and highlight the target.

![](docs/images/search_bar.png)

### 5. Header and Status Bar Actions
- **Reload File**: Click the circular arrow icon next to the file path in the top header to refresh the document from disk.
- **Copy Path**: Click the file path in the top header to copy the full location to your clipboard.
- **Copy Filename**: Click the filename in the bottom status bar to copy just the name of the file.
- **Live Stats**: The status bar displays real-time word count, line count, and your current cursor position (Line/Col).

### 6. Customization and View
- **View Mode**: Click the **Eye** icon in the toolbar to toggle between Edit Mode and a distraction-free View Mode.
- **Zooming**: Use the **A+** and **A-** icons in the toolbar to quickly adjust the view size.
- **Theme**: Switch between Light, Dark, System, and E-paper modes in the View menu.
- **Language**: Choose your preferred interface language from the View menu.
